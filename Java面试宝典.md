# 一.Java的优点

1)为纯面向对象的语言

2)平台无关性

3)Java提供了很多内置的类库,通过这些类库简化了开发人员的程序设计工作,同时缩短了项目开发时间

4)提供了对web应用的开发支持

5)具有较好的安全性和健壮性

6)去除了C++中难以理解容易混淆的特性

# 二.Java与C/C++有什么异同

Java与C/C++都是面向对象语言,都使用了面向对象思想

不同之处:

1)Java为解释型语言,而C/C++为编译型语言

2)Java为纯面向对象语言,所有代码必须在类中实现,除基本数据类型外所有所有类型都是类

C++兼具面向对象和面向过程变成的特点

3)与C/C++相对比,Java没有指针的概念,有效防止操作指针引起的系统问题

4)与C/C++相比,Java不支持多重继承,但是Java引入接口的概念,可以同时实现多个接口

5)在C++语言开发过程中,C++开发人员需要管理对内存的分配,而Java语言提供了垃圾回收器来实现垃圾的代码自动回收,不需要程序显示的管理内存分批
